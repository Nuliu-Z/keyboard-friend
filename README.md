# keyboard-friend · One Press to Trigger Your Commands

## Introduction

This is a simple and practical macro keyboard.
What makes it special?

- **Configuration-friendly** – No additional driver installation required
- **DIY-friendly** – Few parts, easy to source and assemble
- **Cross-platform** – Supports Windows, Linux, and Android

---

## Use Cases

- Enter game cheat codes (the original intention, paying tribute to classic games)
- Quickly type passwords
- One-key combo (theoretically feasible, feel free to try)

---

## Hardware List

| Component | Description |
|-----------|-------------|
| 4×4 I2C Matrix Keyboard | 16-key layout with I2C communication |
| nRF52840 Pro Micro | Main controller board |
| USB Type-C Cable | Power + data transmission |

---

## Quick Start

Two steps, no hassle

1. **Configure** – Edit the configuration file to set the key inputs you need
2. **Press** – Connect the device and press the corresponding key to trigger

---

## Limitations

- Does not support Chinese character input
- Does not support wireless connection (requires wired use)
